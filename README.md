# 🌐 Personal Portfolio Website

This is my **Personal Portfolio Website** built using **React**, **Tailwind CSS**, and **React Router DOM**. It showcases my projects, skills, and contact information in a clean, responsive layout.

## 🚀 Live Demo

🔗 [View Live](https://portfoliosohel.netlify.app)

## 🛠️ Tech Stack

- ⚛️ React.js  
- 🎨 Tailwind CSS  
- 🔗 React Router DOM  

## 📂 Features

- 🏠 Home Section  
- 👤 About Section  
- 💼 Projects Showcase  
- 🛠️ Skills Section  
- 📞 Contact Form  
- 📱 Fully Responsive Design
